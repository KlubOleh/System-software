# 🚀 GPU Control & Monitor Utility

**Курсова робота** **Тема:** Розробка програмного забезпечення для керування відеокартами на базі архітектури NVIDIA.  
**Університет:** НУ "Львівська політехніка"  
**Стек:** C++17/20, WinAPI, NVIDIA NVAPI SDK, Qt/ImGui.

---

## 📌 Про проєкт
Дане програмне забезпечення розроблене для реального моніторингу та тонкого налаштування параметрів графічних адаптерів. Утиліта дозволяє знайти баланс між високою продуктивністю в іграх та акустичним комфортом і безпечними температурами для ноутбуків.

---

## ✨ Основні фічі (Features)

### 📊 Моніторинг та телеметрія
* **Live Clock Tracking:** Відстеження частот ядра та пам'яті в реальному часі.
* **Thermal Dashboard:** Візуалізація температури GPU та точок Hot Spot.
* **VRAM Analyzer:** Контроль використання відеопам'яті.
* **Power Draw Monitoring:** Відображення реального споживання енергії у Ватах (W).

### ❄️ Керування охолодженням
* **Manual Fan Override:** Можливість ручного встановлення швидкості вентиляторів.
* **Intelligent Fan Curve:** Редактор кривої залежності обертів від температури ($T \rightarrow RPM$).

### ⚡ Оптимізація енергоспоживання
* **Power Limit Scaling:** Регулювання лімітів TDP для зниження нагріву (Undervolting).
* **Profile Manager:** Швидке перемикання між режимами:
    * 🤫 **Silent:** Мінімальний шум для роботи/навчання.
    * 🎮 **Gaming:** Оптимальний баланс для ігор.
    * 🔥 **Extreme:** Максимальна потужність.

### 🛠 Системні можливості
* **Session Logging:** Запис показників у `.csv` або `.log` для аналізу стабільності.
* **Tray Integration:** Робота у фоновому режимі з відображенням температури в системному треї.
* **Auto-Reset:** Безпечне повернення до заводських налаштувань при закритті програми.

---

## 🏗 Архітектура модулів
Проєкт реалізовано за модульним принципом:
1. **Hardware Abstraction Layer (HAL):** Пряма взаємодія з `nvapi.dll`.
2. **Telemetry Engine:** Обробка даних у фоновому потоці (1000 мс).
3. **Control Logic:** Математична модель керування частотами та напругою.
4. **UI Layer:** Графічний інтерфейс користувача з підтримкою Dark Mode.
