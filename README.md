# 🚀 GPU Control & Monitor Utility

**Курсова робота** **Тема:** Розробка програмного забезпечення для керування відеокартами на базі архітектури NVIDIA.  
**Платформа:** Windows 10/11 (x64) | **Мова:** C++

---

## 📌 Мета проєкту
Розробка спеціалізованого програмного забезпечення на мові **C++**, яке дозволяє користувачу в реальному часі здійснювати моніторинг показників відеокарти та корегувати її робочі параметри (частоту, ліміти потужності, оберти вентиляторів) для досягнення оптимального балансу між продуктивністю та нагрівом.

Особливо актуально для власників ігрових ноутбуків (наприклад, **Acer Nitro V15**) для запобігання термічному тротлінгу.

---

## 🛠 Архітектура та модулі
Програма побудована за модульним принципом:

### 1. Hardware Abstraction Layer (HAL) 🔌
Посередник між ОС та драйвером відеокарти.
* **NVAPI SDK:** Прямий доступ до внутрішніх регістрів GPU через `nvapi.dll`.
* **Ідентифікація:** Визначення Device ID, версії VBIOS та драйвера.
* **Сенсори:** Зчитування температури ядра, обертів (RPM) та енергоспоживання (TDP).

### 2. Telemetry Engine 📊
Працює у фоновому потоці (частота 1000 мс).
* **Моніторинг:** Відстеження частот (Boost Clock) та використання пам'яті (VRAM).
* **Логування:** Запис даних у `.csv` або `.log` для аналізу стабільності (наприклад, під час гри в PUBG).

### 3. Control & Optimization ⚙️
Активна частина для зміни стану заліза:
* **Power Limit Scaling:** Регулювання TDP (зниження на 10-20% суттєво зменшує температуру).
* **Custom Fan Curve:** Налаштування графіка обертів (наприклад: 50°C → 30%, 85°C → 100%).
* **Profile Manager:** Пресети *Silent, Gaming, Extreme*.

## 🔄 Алгоритм роботи (Main Loop)



1. **Ініціалізація:** Завантаження драйверів та перевірка GPU.
2. **Основний цикл:**
   - Опитування `NvAPI_GPU_GetThermalSettings`.
   - Зчитування частот через `NvAPI_GPU_GetAllClockFrequencies`.
   - Порівняння температури з лімітом (90°C).
   - Коригування кулерів згідно з кривою.
   - Оновлення GUI.
3. **Завершення:** Повернення параметрів до заводських значень при виході.
