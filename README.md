# GPU Control & Monitor Utility 🚀

Курсова робота студента Національного університету "Львівська політехніка".
Програма для моніторингу та керування параметрами відеокарт NVIDIA на C++.

## 📌 Про проєкт
Метою роботи є розробка утиліти, що дозволяє в реальному часі контролювати стан GPU та змінювати його робочі параметри (частоту, Power Limit, кулери).

### Основні модулі:
- **Hardware Abstraction Layer (HAL):** Пряма взаємодія з `nvapi.dll`.
- **Telemetry Engine:** Збір даних (температура, завантаження, RPM) з частотою 1000 мс.
- **Control & Optimization:** Керування кривою вентиляторів та профілями енергоспоживання.

## 🛠 Технології
- **Мова:** C++17/20
- **Бібліотеки:** NVIDIA NVAPI SDK, Qt 6 (GUI)
- **Компілятор:** MSVC (Visual Studio 2022)

## ⚙️ Функціонал
- [x] Моніторинг температури та частот.
- [x] Налаштування кастомної кривої вентиляторів (**Custom Fan Curve**).
- [x] Регулювання **Power Limit** (зниження на 10-20% для Acer Nitro).
- [x] Система профілів: *Silent, Gaming, Extreme*.

## 🚀 Встановлення та запуск
1. Склонуйте репозиторій.
2. Відкрийте проєкт у Visual Studio.
3. Переконайтеся, що NVAPI SDK підключено до проєкту.
4. Запустіть від імені адміністратора (необхідно для доступу до драйвера).
